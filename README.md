# Agenda Facil

Sistema web para negocios de servicos como salao, barbearia, manicure, estetica e petshop.

## Funcionalidades

- Login administrativo.
- Cadastro de negocio.
- Cadastro de servicos com preco e duracao.
- Cadastro de profissionais.
- Agenda diaria com status.
- Agendamento manual pelo dono.
- Link publico para clientes agendarem.
- Mensagem pronta para WhatsApp.
- Persistencia local para demonstracao.

## Como rodar

```bash
node server.js
```

Acesse:

```text
http://127.0.0.1:4180
```

Login de demonstracao:

```text
admin@agenda.local
admin123
```
